File Handling package

This package contains classes that are used to handle files in Java. java.io.File

  1. Create a folder

                        
                        public static void createFolder(String folderPath){
                            File folder = new File(folderPath);
                            if(folder.exists()){
                                System.out.println("Folder already exists");
                            }else{
                                folder.mkdir();
                                System.out.println("Folder created successfully");
                            }
                        }

    Check folder if exists

                        
                        public static void checkFolder(String folderPath){
                            File folder = new File(folderPath);
                            if(folder.exists()){
                                System.out.println("Folder exists");
                            }else{
                                System.out.println("Folder does not exist");
                            }
                        }

    Rename a Folder

                        
                        public static void renameFolder(String folderPath, String newFolderPath){
                            File folder = new File(folderPath);
                            File newFolder = new File(newFolderPath);
                            if(folder.exists()){
                                folder.renameTo(newFolder);
                                System.out.println("Folder renamed successfully");
                            }else{
                                System.out.println("Folder does not exist");
                            }
                        }
